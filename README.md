# Pengumuman: Konten Telah Pindah

> **Status repo:** **Archived / Read-only.**
> Konten **Scoping Review Guide** sekarang menjadi salah satu track di platform
> resmi **SLR Research RPI Institute**.

***

## Pindah ke Platform Baru

Mulai **April 2026**, panduan **Scoping Review (JBI Manual Ch.11 + PRISMA-ScR)**
telah diintegrasikan ke platform pembelajaran terpadu:

### → [https://slr.rumahpublikasi.id](https://slr.rumahpublikasi.id)

Platform baru menyediakan **3 track pembelajaran** dalam satu akun:

| Track | Untuk Siapa |
|---|---|
| **SLR AI** | Workflow chat-based klasik — PICO + extraction + meta-analysis |
| **SLR + AI Cowork** | File-based via Claude desktop — hemat token 40-50%, output `.docx` |
| **Scoping Review** _(eks repo ini)_ | JBI + PRISMA-ScR + PCC + EGM (Evidence Gap Map) |

***

## Mengapa Pindah?

| Fitur | Repo GitBook (lama) | Platform baru |
|---|---|---|
| Akses konten | Public read-only | Login + tracking progress |
| Multi-track | 1 (Scoping Review saja) | 3 (SLR AI / Cowork / ScR) |
| Switch antar track | Tidak ada | Dropdown di sidebar, progress per-track terjaga |
| Tracking modul | Tidak ada | Auto-save per user per modul |
| Dual-reviewer | Manual | Spreadsheet shared + kappa Cohen otomatis |
| Compile manuskrip | Manual paste ke Word | Auto `.md` → `.docx` (Pandoc/Python) |
| Update konten | Push ke GitBook | Real-time deploy via Cloud Run |

***

## Cara Akses Platform Baru

1. Buka **[https://slr.rumahpublikasi.id](https://slr.rumahpublikasi.id)**
2. **Daftar akun** (atau login Firebase Auth Google)
3. Setelah login → halaman **"Pilih Track"** muncul
4. Pilih **Scoping Review** untuk konten yang sebelumnya di repo ini
5. Mulai dari Modul 1 — sidebar otomatis menampilkan 9 modul ScR

***

## Konten yang Dipindahkan

Seluruh 9 modul Scoping Review yang sebelumnya di repo ini telah dipindah ke
platform baru, **versi terbaru dengan workflow Claude cowork file-based**:

* Modul 1 — Fondasi Teori Scoping Review (JBI + PRISMA-ScR)
* Modul 2 — Topik Penelitian (PCC framework)
* Modul 3 — Search Strategy (peer + grey literature)
* Modul 4 — Data Mining + Multi-Source Deduplication
* Modul 5 — Title & Abstract Screening (dual-reviewer + Cohen's kappa)
* Modul 6 — Full-text Acquisition + Auto-Download (Unpaywall + Wayback)
* Modul 7 — Data Charting (iteratif, Levac et al.)
* Modul 8 — Mapping + Evidence Gap Map (EGM)
* Modul 9 — Manuscript Writing PRISMA-ScR + compile `.docx`
